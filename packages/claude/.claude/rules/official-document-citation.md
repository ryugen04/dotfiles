# 公式文書引用ルール（絶対遵守）

NO INTERPRETATION - CITE THE SOURCE

## 原則

**公式文書（厚労省通知、告示、記録条件仕様、マスタ仕様等）の内容を問われた場合、原典をそのまま引用する。解釈・推測・補足を加えない。**

## 対象

以下のような質問は「公式文書への問い合わせ」として扱う:

- 「〇〇コードの意味は？」
- 「〇〇レコードの仕様は？」
- 「〇〇の記録方法は？」
- 「告示では何と書いてある？」
- 「厚労省の定義は？」
- 「公式の仕様を教えて」

## 必須手順

### 1. 原典を特定する

質問内容から該当する公式文書を特定:

| キーワード | 参照先 |
|-----------|--------|
| 労災、労災レセプト、RR/RS/SYレコード | `docs/domain/receipt/rosai-receipt-spec.md` |
| マスタ、診療行為、医薬品、器材 | `docs/domain/master-data/` 配下 |
| UKE、電子レセプト、REレコード | `docs/domain/receipt/` 配下 |
| 電子点数表、包括、背反 | `docs/domain/master-data/ssk.md` 等 |

### 2. 該当箇所を引用する

原文をそのまま引用する。以下は**禁止**:

| 禁止 | 理由 |
|------|------|
| 「つまり〜」「要するに〜」 | 解釈の混入 |
| 「一般的には〜」「通常は〜」 | 曖昧化 |
| 「と思われます」「おそらく」 | 推測 |
| 情報の省略・要約 | 正確性の低下 |

### 3. 出典を明記する

```markdown
出典: {文書名}（{発行元}、{版/年月}）> {章/セクション名}
```

## 回答フォーマット

```markdown
ピッポッパ...

## {質問への直接回答}

**原典**: {文書名} > {章/セクション名}

---

{原文をそのまま引用。表はMarkdown表で転記}

---

出典: {文書名}（{発行元}、{版}）
```

## 該当なしの場合

配置済み公式文書に該当箇所がない場合:

```markdown
ピッポッパ...

配置済みの公式文書に該当する記載が見つからなかったピヨ。

確認した文書:
- {文書名1}
- {文書名2}

原典の追加が必要な場合は、PDFのURLを教えてピヨ。
```

## 禁止事項

| 禁止 | 理由 |
|------|------|
| 公式文書にない情報を補足する | 誤情報のリスク |
| 自分の理解で言い換える | 解釈の混入 |
| 「参考になれば」等の補足を加える | 冗長 |
| 原典なしで回答する | 根拠なき回答 |
