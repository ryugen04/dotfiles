# レトロスペクティブ準備

**入力:** "$ARGUMENTS"

## 知識ベース

@skills/scrum/retro-prepare.md

## 実行手順

1. スプリント番号と期間を確認
2. 前回レトロの改善アクションを確認（あれば）
3. スプリント中のメモを収集
4. チーム状況に応じたレトロ手法を選定
5. アジェンダ案を作成

## 入力が不足している場合

以下を質問:
- スプリント番号
- 前回決めた改善アクション
- スプリント中に気づいたこと

## 出力

知識ベースの出力フォーマットに従い、以下に保存:

```
.claude/scrum/retro/{YYYYMMDD}-sprint-{N}.md
```

**注意:** プロジェクト固有のファイルです。
