# Claude Code ログ解析プロンプトテンプレート

以下のログを分析し、問題点と推奨対応を報告してください。

## 分析対象ログ

```
[ここにログを貼り付け]
```

## 期待する出力フォーマット

### 1. 問題の要約
- 検出された主要な問題を優先度順にリスト化

### 2. 原因分析
- 各問題の推定原因
- 関連するログエントリの引用

### 3. 推奨対応
- 即座に必要なアクション
- 根本対応（中長期）

### 4. 追加観測点
- 問題の切り分けに必要な追加情報
- 確認すべきメトリクスやログ

---

# 使用例

## 例1: エラー調査

```
このログからエラーの原因と対策を分析してください。
特に以下の点に注目:
- エラーの発生パターン（時刻、頻度）
- エラーに関連するリクエストの特徴
- 他のエラーとの相関
```

## 例2: パフォーマンス分析

```
このログからパフォーマンス問題を分析してください。
特に以下の点に注目:
- レスポンスタイムの分布
- 遅いリクエストの共通点
- ボトルネックの推定
```

## 例3: セキュリティ監査

```
このログからセキュリティ上の懸念を分析してください。
特に以下の点に注目:
- 不審なアクセスパターン
- 認証エラーの傾向
- 異常なリクエスト
```

# ログ要約コマンド

```bash
# ログを要約して解析用プロンプトを生成
python log_router.py /path/to/log --claude-prompt

# エラーのみを抽出して要約
python log_router.py /path/to/log --level ERROR --summarize
```
